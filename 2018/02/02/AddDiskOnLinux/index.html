创建分区挂载目录

# mkdir /mnt/disk01

挂载新分区

# mount /dev/sdb1 /mnt/disk01/

设置开机自动挂载磁盘

# vim /etc/fstab

/dev/sdb1    /mnt/disk01    ext4    defaults    0 0

说明:

第一列是设备名
第二列是挂载点（挂载目录）
第三列是要使用的文件系统
第四列是挂载参数，使用默认即可（defaults）
第五列是dump备份参数，1：每天备份，2：不定期备份，0：不备份
第六列是扇区检验参数，1：最早检验，2：标识为1的设备检验完之后检验，0：不检验
