CentOS7下挂载新磁盘

虚拟机也好，物理机也罢，在CentOS的使用过程中都有可能会出现需要增加磁盘的情况，这篇文章就是介绍如何为新磁盘进行分区以及挂载操作。

所有操作都在root用户下进行。

查看当前磁盘使用状况

# df -h

列出现有的磁盘及分区信息

# fdisk -l

对新磁盘 /dev/sdb 进行分区操作

# fdisk /dev/sdb

再次查看磁盘分区信息

# fdisk -l

格式化新分区 /dev/sdb1

这里使用ext4文件系统
# mkfs.ext4 /dev/sdb1

创建分区挂载目录

# mkdir /mnt/disk01

挂载新分区

# mount /dev/sdb1 /mnt/disk01/

查看磁盘使用情况

# df -h

设置开机自动挂载磁盘

# vim /etc/fstab

说明:

第一列是设备名
第二列是挂载点（挂载目录）
第三列是要使用的文件系统
第四列是挂载参数，使用默认即可（defaults）
第五列是dump备份参数，1：每天备份，2：不定期备份，0：不备份
第六列是扇区检验参数，1：最早检验，2：标识为1的设备检验完之后检验，0：不检验
